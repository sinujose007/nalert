Nalert Alert Box Plugin.

== Description ==

Nalert lets you enter a notification message from its settings page by selecting single or multiple custom post types to display the alert box.
For notification Nalert using a third party jquery plugin.

Major features in Nalert include:

* Single settings for multiple page.
* Activate when a notification is ready for end users.

== Installation ==

Upload the Nalert plugin to your blog, Activate it, then got to settings page and submit rquired Information.

You're done!
